
# SCOTUS RSS + Summaries

This repo scrapes recent U.S. Supreme Court decisions from Cornell LII, builds an RSS feed (`feed.xml`), and maintains a second RSS feed (`summary.xml`) with AI-generated case summaries. New cases are summarized automatically and appended.
